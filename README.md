# 🐾 Mi API Wrapper (YouTube & Animals)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Node.js Version](https://img.shields.io/badge/node-%3E%3D14.0.0-brightgreen)](https://nodejs.org/)
[![Type](https://img.shields.io/badge/Type-Hybrid%20(ESM/CJS)-blue)](#)

Un wrapper potente y ligero para obtener imágenes de animales y descargar audio de YouTube en MP3. Diseñado para ser compatible con **CommonJS** y **ES Modules**.

---

## ✨ Características
* **Híbrido:** Funciona con `import` y `require`.
* **Seguro:** Código ofuscado y optimizado.
* **YouTube Scraper:** Conversión automática a MP3 con sistema de espera (*polling*).
* **Animal API:** Acceso rápido a múltiples endpoints de imágenes de animales.
